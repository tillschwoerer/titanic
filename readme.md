# Titanic

This repo is for practicing Git and Github.

We want to cover the basic git workflow today.

- `git init`
- `git add`
- `git commit `
- `git push`

### Collaboration Worflows

- Feature Branch Workflow: scenario with write access to central repo
- Forking Workflow: large open source scenario without write acess

In collaboration work it is possible to run into merge conflicts if multiple people work on the same file (and lines of this file) at the same time.

In this case, we need to manually resolve the conflict - either on Github or locally.
